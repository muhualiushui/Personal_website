# Project Structure

Use one folder per project under `projects/`.

Recommended layout:

- `projects/<project-slug>/index.html`
- `assets/projects/<project-slug>/...`

Naming rules:

- Use lowercase kebab-case slugs.
- Keep one canonical slug for both the page folder and asset folder.
- Put shared files in `assets/shared/`.
- Put research-only documents in `assets/research/`.
